# Gemini Streamlit

### Dependencies

```
pip install streamlit langchain langchain_community langchain_google_genai
```

### 사용법

-   api_key 변수에 Gemini API key를 입력해주세요.

-   질문은 20글자 이상 입력해주세요.(띄어쓰기 포함)

-   질문창에 "초기화"를 입력해 상담을 새로 시작할 수 있습니다.
